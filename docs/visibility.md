# Relay Console visibility plan

Use this file as the working checklist for improving Relay Console discoverability.

## Primary description

Use this phrase consistently:

> Local, offline, single-file HTML tool for manually relaying conversations between multiple AI assistants.

Shorter variants:

- Manual AI relay for ChatGPT, Claude, Gemini, and other assistants.
- A copy-paste relay console for comparing AI chatbots without APIs.
- You are the wire between the models. Relay Console handles the structure.

The phrase to own is **manual AI relay**. Secondary phrases are **copy-paste AI relay**, **offline AI assistant relay**, **multi-assistant relay tool**, **local-first AI workflow tool**, **no-API multi-model workflow**, and **single-file AI orchestration tool**.

Do not position the project mainly as an autonomous multi-agent framework. Relay Console is a human-in-the-loop relay tool for people using consumer chatbot subscriptions.

## README first screen

The first screen of the README should include:

- local
- offline
- single-file HTML
- manual relay
- AI assistants
- ChatGPT
- Claude
- Gemini
- no API keys
- no backend
- no build step

The first screen should also say that Relay Console does not call model APIs, does not automate chatbot tabs, does not inspect browser tabs, does not run a backend server, and does not upload user prompts or answers.

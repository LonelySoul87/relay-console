# Relay Console

A local, offline, single-file HTML tool for manually relaying conversations between multiple AI assistants.

You copy prompts out. You paste answers back. Relay Console keeps the multi-model conversation structured.

## How it works

1. Write one question and choose the assistants.
2. Relay Console builds each turn's prompt with context and role framing.
3. Copy the prompt into the selected chatbot.
4. Paste the answer back.
5. Continue through the relay.
6. Merge the results into one final answer.

## Why manual relay?

Most multi-model tools assume API access. Relay Console is for people who already pay for consumer chatbot plans and want a structured, auditable workflow without separate API billing or hosted orchestration.

The point is not to remove the human from the loop. The point is to make the loop less chaotic.

## Privacy and offline

Relay Console runs locally in your browser. It does not require API keys, an account, a server, or a build step.

It does not upload prompts, answers, presets, or sessions. The user decides what gets sent to each AI provider.

## What it is not

- Not an API agent framework.
- Not a browser extension.
- Not a hosted AI workspace.
- Not a tab automation tool.

# Announcement copy

Relay Console is a local, offline, single-file HTML tool for manually relaying conversations between multiple AI assistants.

It is intentionally manual. You copy prompts out, paste answers back, and Relay Console keeps the transcript, model roles, context, prompt framing, and final synthesis organized.

No API keys. No backend. No browser extension. No hosted storage.

## Longer version

I built Relay Console because I already use several consumer AI assistants, but I did not want to recreate that setup through API billing just to make them critique and compare one another.

Relay Console is a local single-file HTML app for a structured copy-paste relay. It prepares each model's prompt, keeps the transcript organized, preserves the answers, and helps merge the results into one final response.

It does not call model APIs, automate chatbot tabs, or run a backend. The user remains the wire between the models.

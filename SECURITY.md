# Security policy

## Reporting a vulnerability

Please do not open a public GitHub issue for a suspected security or privacy vulnerability.

Email [lonelysoul.projects@gmail.com](mailto:lonelysoul.projects@gmail.com) with:

- the affected Relay Console version
- steps to reproduce the issue
- the browser and operating system used
- the potential security or privacy impact

Do not include real private conversations, credentials, tokens, or other sensitive personal data in the report. A minimal synthetic example is preferred.

## Scope

Relay Console is a local, single-file browser application. Reports concerning unexpected network requests, unsafe import behavior, data loss, prompt-boundary handling, or exposure of locally stored session data are especially relevant.

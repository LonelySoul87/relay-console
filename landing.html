Relay Console

A local, offline, single-file HTML tool for manually relaying conversations between ChatGPT, Claude, Gemini, and other AI assistants.

You copy prompts out. You paste answers back. Relay Console keeps the multi-model conversation structured and easier to synthesize.

Open Relay Console Download latest release View on GitHub

Local and offline

Open the saved HTML file in your browser. The tool itself does not need a server or account.

Human-in-the-loop

You decide what gets copied, pasted, trimmed, skipped, or synthesized at every step.

No API orchestration

Relay Console does not call model APIs or automate chatbot tabs. It organizes manual relay work.

How it works

  1. Write one question and choose the assistants.
  2. Relay Console builds a prompt with role framing and relevant context.
  3. Copy the prompt into the selected chatbot, then paste the answer back.
  4. Continue through debate or blind-comparison mode, then synthesize the results.

What it is not

  • Not an autonomous agent framework.
  • Not a browser extension.
  • Not a hosted AI workspace.
  • Not a tab automation tool.
